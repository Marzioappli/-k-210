Alles über Cloud's

Was ist eine Cloud?
Nein es hat nichts mit einer Wolke, die man im Himmel sieht zu tun!
Wenn man etwas auf eine Cloud ladet,speichert man Daten auf einem entfernten Server.
Dabei werden die verschiedenen Daten von einem Gerät über das Internet auf den Server eines Cloud Anbieters geladen.
Nachher können die verschiedenen Dateien mit diesem oder anderen Geräten wieder abgerufen werden.
Dass heisst, man kann seine Dateien von verschiedenen Orten und verschiedene Geräte bearbeiten.
Am Ende spart man sehr viel Speicherplatz.
Wann brauche ich eine Cloud?
Dokumente, Fotos, Filme und auch Anwendungen muss man nicht nur auf der Festplatte von Geräte zu Hause bearbeiten und speichern.
Sie können auf entfernte Server ausgelagert werdenCloud's

Was kann man auf einer Cloud speichern?

  • private Fotos
  • Videos
  • Textdateien
  • private Dateien

Risiken einer Cloud!
Mann muss sicher sein, dass man seine Daten wieder erhält, denn wenn man sein Login etc. bekannt gibt,
können diese gestohlen werden. Deshalb ist es wichtig ein Backup-Szenario in der Cloud zu machen!
